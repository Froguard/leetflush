# 常见排序介绍

> 各大排序的实现，以及优劣，空间和时间负责度的最优和最劣情况

## 九大排序的分类概览

结论：

1. `堆排序`，`归并排序`，最综合，且稳定：最快，最慢，都是 O(NlogN)
2. `快速排序` 综合性次之，最快 O(NlogN)，最慢 O(N^2)
3. `冒泡排序`、`插入排序` 保持在 最慢 O(N^2)， 最快 O(N)
4. `选择排序` 最慢，速度稳定在 O(N^2)
